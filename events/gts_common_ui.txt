# 殖民地发展方式：
# 巨娘帝国：
# 每个星球增加决议可以缩小星球上的居民（发展缓慢但稳定）
# 天体，泰坦，巨大生成缩小人，高大，正常，矮小生成缩小城市，微小，微生物，分子生成缩小星球
# 这三类资源可以用于发展身体殖民地，殖民地会显示这三类资源的数量
# 女神：是殖民地的统治阶层，负责规划殖民地的发展，根据殖民地位置与女神数量的不同提供领袖BUFF
# 女仆：是殖民地的专家阶层，管理殖民地并将其作为实验材料，生成进阶资源（合金，消费品，科研，传统，遗迹）
# 小人：是殖民地的劳工阶层，负责生产资源维护殖民地，生成基础资源（食物，矿物，能量）
# 小人帝国：
